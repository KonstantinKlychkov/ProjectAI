# ProjectAI — мультиагентная система управления проектами

## Назначение навыка

Используй этот навык, когда работаешь в репозитории ProjectAI: проектирование агентов, требования, исследования, код или документация. Сохраняй терминологию и структуру проекта из бизнес-требований.

## Структура проекта

| Путь | Содержание |
|------|-------------|
| `Business_requirements/business_requirements.md` | Бизнес-требования к МАС: проблемы, решения через агентов, функциональные и нефункциональные требования |
| `Researches/` | Исследования по MAS for PM (ChatGPT, Grok, Gemini, DeepSeek): архитектура, технологии, практики |

## Ключевые агенты (из требований)

- **Стратегический уровень:** агент планирования (декомпозиция целей), агент риск-менеджмента, агент анализа Lessons Learned.
- **Процессы:** агент управления ролями, агент мониторинга проектов, агент уведомлений, агент координации зависимостей, агент приоритизации задач, агент прогнозирования сроков, агент адаптации планов, агент эскалации проблем, агент моделирования изменений.
- **Операции:** агент координации (поиск данных, синхронизация), агент управления загрузкой ресурсов, агент автоматизации документации, агент генерации отчётов.
- **Компоненты (сводная таблица):** агент стратегического планирования, агент управления задачами, агент контроля метрик и KPI, агент риск-менеджмента, агент интеграции.

Подробный список агентов, решаемых проблем и терминология — в [reference.md](reference.md).

## Правила при работе в проекте

1. **Терминология:** используй «МАС» / «мультиагентная система», «агент» для программных агентов; «управление проектами», «декомпозиция», «риск-менеджмент», «эскалация» в значении из бизнес-требований.
2. **Требования:** при предложении функций или агентов сверяй с `Business_requirements/business_requirements.md` — не противоречь описанным проблемам и решениям.
3. **Исследования:** при выборе технологий или архитектуры учитывай материалы в `Researches/` (JADE, SPADE, FIPA, протоколы, масштабируемость).
4. **Язык:** ответы пользователю — на русском, если иное не указано; код и идентификаторы — на английском.
