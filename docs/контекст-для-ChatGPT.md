# Контекст проекта ProjectAI для ChatGPT

**Как использовать:** скопируй блок «Контекст для вставки» ниже в начало диалога с ChatGPT или в раздел «Инструкции» (Instructions) при создании Custom GPT. Тогда ответы будут учитывать терминологию и структуру проекта.

---

## Контекст для вставки

```
Проект: ProjectAI — мультиагентная система (МАС) для управления проектами. Цель: повышение эффективности через цифровизацию процессов и data-driven управление.

Терминология: МАС = мультиагентная система; агент = программный компонент с автономным поведением и чёткой ролью; декомпозиция = разбиение целей на подцели и измеримые задачи; риск-менеджмент, Lessons Learned, эскалация — в значении управления проектами.

Ключевые агенты (по уровням):
- Стратегический: агент планирования (декомпозиция целей), агент риск-менеджмента, агент анализа Lessons Learned.
- Процессы: агент управления ролями, агент мониторинга проектов, агент уведомлений, агент координации зависимостей, агент приоритизации задач, агент прогнозирования сроков, агент адаптации планов, агент эскалации проблем, агент моделирования изменений.
- Операции: агент координации (поиск данных, синхронизация систем), агент управления загрузкой ресурсов, агент автоматизации документации, агент генерации отчётов.
- Сводные компоненты: агент стратегического планирования, агент управления задачами, агент контроля метрик и KPI, агент риск-менеджмента, агент интеграции.

Документы в репозитории:
- Бизнес-требования: Business_requirements/business_requirements.md
- Исследования MAS for PM: Researches/ (ChatGPT, Grok, Gemini, DeepSeek)
- Справочник агентов и терминов: .cursor/skills/projectai-mas-pm/reference.md

Отвечай с учётом этой терминологии и структуры. Код и идентификаторы — на английском; пояснения и комментарии для пользователя — на русском, если не указано иное.
```

## Рекомендации по работе с ChatGPT

1. **Новый диалог по проекту** — вставь контекст выше в первое сообщение или сразу после приветствия.
2. **Custom GPT** — создай своего GPT (My GPTs → Create), вставь этот контекст в Instructions. Можно добавить: «Пользователь работает над проектом ProjectAI; при ответах опирайся на загруженные файлы и контекст».
3. **Исследования и документы** — при вопросах по архитектуре или агентам прикрепляй или вставляй фрагменты из `business_requirements.md` и файлов из `Researches/`.
4. **Черновики** — проси формулировать разделы требований, описания агентов или протоколы в том же стиле и терминологии, что в репозитории; результат сохраняй в соответствующие папки проекта.
