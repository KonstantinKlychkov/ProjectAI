# Performance Domain: Resources

## 1. Введение в домен

**Resources** — управление ресурсами проекта: планирование управления ресурсами, оценка ресурсов активностей; приобретение ресурсов, развитие команды, управление командой; контроль ресурсов.

**Связь с Focus Areas** ([reference/process-groups-to-domains.md](reference/process-groups-to-domains.md)): Planning (5.15, 5.16), Executing (6.4, 6.5, 6.6), Monitoring & Controlling (7.8).

**Процессы домена Resources:**

- 5.15 Plan Resource Management
- 5.16 Estimate Activity Resources
- 6.4 Acquire Resources
- 6.5 Develop Team
- 6.6 Manage Team
- 7.8 Control Resources

## 2. Процессы домена

**Детали из PMI Guide (Stage 28):**

- **Tools & Techniques:** Expert judgment; Data representation (Hierarchical charts, RAM, Text-oriented); Organizational theory; Meetings.
