# Performance Domain: Finance

## 1. Введение в домен

**Finance** — управление стоимостью проекта: планирование управления стоимостью, оценка затрат, определение бюджета; контроль затрат.

**Связь с Focus Areas** ([reference/process-groups-to-domains.md](reference/process-groups-to-domains.md)): Planning (5.11–5.13), Monitoring & Controlling (7.6).

**Процессы домена Finance:**

- 5.11 Plan Cost Management
- 5.12 Estimate Costs
- 5.13 Determine Budget
- 7.6 Control Costs

### 5.12 Estimate Costs (Planning)

**Детали из PMI Guide (Stage 25):**  
- **Inputs:** PM plan (Cost, Quality, Scope baseline); Project docs (Lessons learned, Schedule, Resource requirements, Risk register); EEF; OPA.

### 5.13 Determine Budget (Planning)

**Детали из PMI Guide (Stage 26):**  
- **Inputs:** PM plan (Cost, Resource mgmt, Scope baseline); Project docs (Basis of estimates, Cost estimates, Schedule, Risk register); Business docs; Agreements; EEF; OPA.

- **Data flow:** связи с 6.9, 7.6; формирует Cost baseline.

### 7.6 Control Costs (Monitoring & Controlling)

**Детали из PMI Guide (Stage 53):**  
- **Inputs:** PM plan (Cost mgmt, Cost baseline, Performance measurement baseline); Project docs (Lessons learned); Project funding requirements; Work performance data; OPA.  
- **Outputs:** Work performance information; Cost forecasts; Change requests; PM plan and Project documents updates.  
- **Tools & Techniques:** Expert judgment; Data analysis (Earned value, Variance, Trend, Reserve); To-complete performance index; PMIS.
