# Performance Domain: Governance

## 1. Введение в домен

**Governance** — управление жизненным циклом проекта: инициация (устав), интеграция планов, исполнение работ, мониторинг и контроль, интегрированное управление изменениями, закрытие проекта или фазы.

**Связь с Focus Areas** ([reference/process-groups-to-domains.md](reference/process-groups-to-domains.md)): Initiating (4.1), Planning (5.1), Executing (6.1), Monitoring & Controlling (7.1, 7.2), Closing (8.1).

**Процессы домена Governance:**

- 4.1 Develop Project Charter
- 5.1 Develop Project Management Plan
- 6.1 Direct and Manage Project Work
- 7.1 Monitor and Control Project Work
- 7.2 Perform Integrated Change Control
- 8.1 Close Project or Phase

**Детали из PMI Guide (Stage 12):**
