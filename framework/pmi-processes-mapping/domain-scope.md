# Performance Domain: Scope

## 1. Введение в домен

**Scope** — управление содержанием проекта: планирование управления содержанием, сбор требований, определение содержания, создание WBS; валидация и контроль содержания.

**Связь с Focus Areas** ([reference/process-groups-to-domains.md](reference/process-groups-to-domains.md)): Planning (5.2–5.5), Monitoring & Controlling (7.3, 7.4).

**Процессы домена Scope:**

- 5.2 Plan Scope Management
- 5.3 Collect Requirements
- 5.4 Define Scope
- 5.5 Create WBS
- 7.3 Validate Scope
- 7.4 Control Scope
